#### 브랜치 사용하기

```
//브랜치 생성
git branch [브랜치명]
//브랜치로 전환하기 (checkout : 내가 사용할 브랜치 지정)
git checkout [브랜치명]

//Github Repository(origin)에 add, commit, push
git add .
git commit -m "커밋메시지"
git push origin product
```

옵션을 사용하지 않고 branch명령어를 사용하면 브랜치 전체 목록을 확인할 수 있다.
```
git branch
```
