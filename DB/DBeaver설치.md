

### DBeaver란

- **무료 데이터베이스 관리 툴.**
- open source
- 멀티 플랫폼: Mac OS, Linux, Windows
- JDBC 드라이버를 통한 데이터베이스 통신
- 다양한 DB지원: MySQL, MariaDB, SQLite, Oracle ...
