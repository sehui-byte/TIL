
## TypeScript  
TypeScript는 javascript의 슈퍼셋이라고 한다.  
자바스크립트에 **타입을 지정**할 수 있다.

![image](https://user-images.githubusercontent.com/64109506/113795735-835a1f00-9788-11eb-95a8-6c7588ce3f5c.png)

```
//전역으로 typescript모듈 설치
npm i typescript -g

//tsconfig.json생성
tsc --init

//ts파일 컴파일
tsc
```
